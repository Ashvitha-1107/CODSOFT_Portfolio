# Personal Portfolio Website

## 📌 Project Overview

This is a responsive Personal Portfolio Website developed using **HTML5** and **CSS3** as part of the **CodSoft Web Development Internship – Level 1 Task 1**.

The website showcases my profile, skills, projects, resume, and contact information in a clean and user-friendly layout.

---

## ✨ Features

- Responsive Design
- Header Section
- About Me Section
- Skills Section
- Projects Showcase
- Resume Download Link
- Contact Information
- Footer with Copyright
- Modern UI with Colorful Styling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
